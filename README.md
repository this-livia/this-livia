## 🚀 Sobre mim

Sou cientista de dados sênior com 8 anos de experiência em análise, modelagem e transformação de dados em valor de negócio. Possuo formação sólida em Estatística pela Universidade Federal do Ceará (UFC) e Mestrado em Modelagem e Métodos Quantitativos (UFC), o que me proporciona uma base teórica robusta em métodos estatísticos e quantitativos avançados.

Durante minha trajetória, participei de projetos estratégicos desenvolvendo modelos preditivos, soluções de BI e pipelines de dados escaláveis que impactaram diretamente nos resultados organizacionais. Tenho experiência prática com ferramentas modernas de engenharia de dados, análise exploratória (EDA), modelagem estatística e visualização de dados.

Minha especialidade abrange desde o tratamento e transformação de dados em SQL (Athena, MySQL, SQL Server) até a construção de dashboards interativos em Power BI e Tableau. Trabalho constantemente com Python e R para análise avançada, Airflow para orquestração de workflows, e cloud computing para escalabilidade.

Sou movido pela inovação e pela busca contínua de conhecimento. Acredito que dados bem tratados e insights bem apresentados são ferramentas poderosas para transformar ideias em decisões estratégicas de impacto.
